<h1 align="center">社区二手交易小程序</h1>

### Project directory

<pre>
packages
├── client              社区二手交易小程序
└── cloud               云开发
</pre>

### Install

Install `node_modules`

```sh
yarn install || npm install
```

### Start project

```sh
yarn dev || npm run dev
```

### Build project

```sh
yarn build || npm run build
```

### CloudBase

开发完成后，需运行下面命令

```sh
yarn serve
```

之后在进行上传。
